DAR AL TAWḤID – CLEAN REBUILD UPLOAD

Das ist ein kompletter sauberer Neuaufbau der Website-Dateien.

Wichtig:
Alte Testdateien können in GitHub bleiben, sie stören nicht, solange index.html, posts.json und service-worker.js korrekt sind.
Wenn du alles sauber machen willst, lösche alle alten Testdateien und alten Ordner – außer diese Dateien aus diesem Paket.

Diese Dateien müssen im Hauptordner liegen:
index.html
posts.json
manifest.json
service-worker.js
CNAME
app-icon-192.png
app-icon-512.png
apple-touch-icon.png
favicon-32.png
favicon-16.png
app-logo-original.png
app-share-image.png
logo-black.png
logo-blue.png
logo-cream.jpg

Hochladen:
ZIP entpacken.
Nur die Dateien innen drin hochladen/ersetzen.
Nicht den ganzen Ordner hochladen.
Commit changes klicken.

Dann öffnen:
https://dar-al-tawhid.de/?v=clean-rebuild-1

Falls alte Ansicht erscheint:
iPhone/Safari: Einstellungen → Safari → Verlauf und Websitedaten löschen.
