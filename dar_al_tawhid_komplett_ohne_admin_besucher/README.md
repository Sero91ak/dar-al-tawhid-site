# DAR AL TAWHID

Professionelle Karten-Version ohne sichtbaren Admin-/Bearbeiten-Button auf der Besucher-Seite.

- Startseite zeigt nur Karten.
- Vollständiger Beitrag öffnet sich erst nach Klick auf „Beitrag öffnen“.
- Beiträge liegen als Markdown-Dateien in `content/posts/`.
- Admin-Ordner ist technisch vorhanden, aber nicht öffentlich auf der Startseite verlinkt.

Netlify:
Build command: `npm run build`
Publish directory: `.`
