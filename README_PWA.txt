DAR AL TAWHID – PWA App-Version

Diese Version macht die Website installierbar als App.

Dateien hochladen/ersetzen im GitHub-Hauptordner:
- index.html
- manifest.json
- service-worker.js

Optional, falls noch nicht vorhanden:
- logo-black.png
- logo-blue.png
- logo-cream.jpg

Wichtig:
Die Website muss über HTTPS laufen. GitHub Pages erfüllt das.
Nach dem Upload die Seite 1–2 Mal neu laden, dann erscheint im Browser oft „App installieren“.
