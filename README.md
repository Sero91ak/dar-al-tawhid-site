# DAR AL TAWHID Site

Diese Website nutzt GitHub + Netlify + Decap CMS.

## Dateien

- `index.html` öffentliche Website
- `admin/` Admin-Bereich
- `content/posts/` Beiträge als Markdown
- `scripts/build.js` erstellt `posts.json`
- `netlify.toml` Netlify Build-Konfiguration

## Netlify Build

Build command:

```bash
npm run build
```

Publish directory:

```bash
.
```
